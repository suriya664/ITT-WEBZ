INSTALLATION:
1. Extract the template files to your web server directory
2. Ensure all CSS and JS files are properly linked
3. Replace placeholder images with your own images
4. Update contact information and content
5. Configure your server for proper routing

CUSTOMIZATION:
- Colors: Edit CSS variables in assets/css/style.css
- Fonts: Update Google Fonts import in style.css
- Content: Edit HTML files in pages/ directory
- Images: Replace images in assets/img/ directories
