UPLOAD METHODS:
---------------

METHOD 1: Using FTP Client (FileZilla, WinSCP, etc.)
1. Connect to your FTP server
2. Navigate to public_html or www folder
3. Upload entire "acybercloud-template" folder
4. After upload, rename "acybercloud-template" to your desired name (optional)

METHOD 2: Using cPanel File Manager
1. Login to cPanel
2. Open File Manager
3. Navigate to public_html
4. Upload the entire "acybercloud-template" folder
5. Extract if uploaded as ZIP

METHOD 3: Using Command Line (SSH)
1. SSH into your server
2. cd to public_html
3. Upload files using SCP or SFTP

IMPORTANT CONFIGURATION:
------------------------

OPTION A: If index.html is in pages/ folder:
- Your homepage URL will be: yourdomain.com/pages/index.html
- Or set up URL rewriting to make it: yourdomain.com/

OPTION B: Move index.html to root (RECOMMENDED):
1. Copy index.html from pages/ to root (acybercloud-template/)
2. Update all links in index.html:
   - Change "about.html" to "pages/about.html"
   - Change "services.html" to "pages/services.html"
   - etc. for all internal links

OPTION C: Use .htaccess (Apache servers):
Create .htaccess in root with:
---
RewriteEngine On
RewriteRule ^$ pages/index.html [L]
RewriteRule ^([^/]+)/?$ pages/$1.html [L]
---
